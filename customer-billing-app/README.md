# Customer Invoicing and Billing App

Backend: .NET Web API
Frontend: Java (Spring Boot + Thymeleaf)

## Features
- Customer account management
- Bill generation and history
- Payment processing (mocked)
- Usage summary and downloadable invoices
- Notifications (mock)
- Admin dashboard

## Structure
- `/api` - .NET Web API backend
- `/ui` - Java Spring Boot + Thymeleaf frontend
